# Групповой проект на тему "Использование видеокарты для обработки изображений"
## Цель проекта - создать приложение, принимающий на вход изображение и выдающие на выход тоже изображение с повышенной контрастностью. Взаимодействие с пользователем должно происходить при помощи графического интерфейса

## Библиотеки и фреймворки использованные в проекте
Проект сделан на **C# .NET 8.0** \
Графический интерфейс сделан на **Avalonia**
### NuGet пакеты:
* **CommunityToolkit.Mvvm**
* **Microsoft.Entity.Framework.Sqlite**
* **Newtonsoft.Json**
* **ReactiveUI**
* **LiveChartsCore.SkiaSharpView.Avalonia**

## Возможности приложения
* Загрузка изображение и его конвертация
* Сохранение конвертированного изображения
* Сохранение данных о процессе конвертации в базе дыннх Sqlite
* Экспорт базы данных в Json
